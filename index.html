Addendum

Problem Solved: In this assignment, I created a web page with a random "fun fact" generator in the main content area. The f eature, triggered by a button click, was developed with JavaScript and uses an array of facts to display one randomly each time the button is pressed. This interactive element helps demonstrate how JavaScript can enhance user engagement within a structured HTML and CSS layout.

Solution Approach:

  • HTML Structure: I used HTML5 semantic elem ents to create a structured page layout following the Holy Grail Layout des ign. Each section—header, sidebar, main content, advertisement, and footer— is clearly defined for easy maintenance and readability.
  • CSS Styling with Grid Layout: The layout w as constructed using CSS Grid, providing flexibility and ensuring all secti ons align consistently across different screen sizes. CSS properties were a djusted to maintain proportional scaling, particularly for the advertisemen t section, to ensure it remains visually balanced within the layout.
  • JavaScript Interactivity: I used JavaScrip t to add a "Get Fun Fact" button, which fetches a random fact from an array . Each button click triggers an event listener to update the displayed text dynamically, allowing users to engage with the page interactively.
  • Footer with Last Modified Script: I added a script in the footer to automatically display the date the page was last updated, as required by the assignment.

Challenges and Solutions:

  • Image Sizing for Advertisement: Initially, the advertisement image was too large and disrupted the layout. To solve th is, I limited its size using CSS `max-width` and `height` properties to ens ure it scales appropriately within the grid layout.
  • JavaScript Event Listener Issue: When I fi rst added the JavaScript button functionality, the event listener didn’t wo rk as expected. The issue was due to the script loading before the DOM was fully available. Adding the `defer` attribute to the script tag ensured tha t the script waited for the DOM to load, resolving the issue.
  • Responsive Design Adjustments: Making the layout responsive required several adjustments, especially for the sidebar and advertisement sections. Media queries and CSS Grid adjustments were use d to maintain consistent alignment across screen sizes.
